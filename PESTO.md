## Start the runtime

```bash
export DESIRED_VERSION=0.0.1
git clone zerrpo voila/
cd voila/


git checkout ${DESIRED_VERSION}

cd runtime/
pnpm i && pnpm start
# > Then you get the QR Code in the terminal

```

* Install the "Expo Go" Mobile app, and scan the QR code
